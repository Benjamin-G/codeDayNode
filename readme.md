# Global Day of Coderetreat 2018

This is a simple template I made for Global Day of Coderetreat 2018.  It uses node and mocha for TTD.

## Installation

Use the package manager [npm](https://www.npmjs.com/get-npm) to install.

```bash
npm install 
```
